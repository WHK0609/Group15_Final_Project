# Group15_Final_Project


RNA-Seq 数据分析及生物学信息可视化研究

本仓库包含处理组（Treated）与对照组（Control）样本的RNA-Seq数据分析及生物学信息可视化完整流程、结果解读，旨在挖掘处理因素对基因表达的调控作用，为相关分子机制研究提供数据支撑与分析范式。

📌 研究背景

RNA测序（RNA-Seq）是转录组学研究的核心技术，凭借高通量、高分辨率优势，广泛应用于基因表达定量、差异表达分析、转录本结构鉴定等领域，为解析基因调控机制、疾病发病机理及环境胁迫响应提供海量转录组数据。

随着测序技术发展，RNA-Seq数据规模持续扩大、结构愈发复杂，如何从海量原始数据中挖掘有价值的生物学信息成为研究瓶颈。生物学信息可视化作为数据与生物学解读的桥梁，通过将抽象数据转化为火山图、热图、PCA图等直观图形，可清晰呈现样本间表达差异、基因表达模式及聚类关系，降低解读难度，为后续功能富集、调控网络构建提供前期支撑。

本研究聚焦处理组与对照组RNA-Seq数据，通过系统分析与标准化可视化流程，挖掘处理因素的基因调控作用，为同类研究提供参考。

🔍 研究内容

2.1 数据预处理与质量控制

对原始RNA-Seq测序数据进行严格质量过滤，去除低质量reads、接头序列及污染序列；通过质量评估工具验证过滤后数据的碱基质量、GC含量等指标，确保分析可靠性。

2.2 基因表达定量与标准化

基于参考基因组，通过序列比对或组装将清洁reads定位到目标基因区域，计算各样本基因表达量（FPKM/TPM值）；对表达量数据标准化，消除测序深度差异，为样本间比较奠定基础。

2.3 样本关联性与整体表达分析

计算样本间表达量相关系数，构建相关性矩阵，分析组内样本重复性与组间差异；统计两组整体基因表达水平，探究处理因素对全局基因表达的影响趋势。

2.4 差异表达基因筛选与验证

采用DESeq2、edgeR等工具，以|log2FC|≥1、padj<0.05为阈值，筛选两组间差异表达基因（DEGs）；统计差异基因数量，明确上调、下调基因分布特征。

2.5 生物学信息可视化分析

构建多维度可视化图表：

- PCA图：展示样本整体分离趋势，验证组间差异显著性；

- 火山图：呈现基因表达倍数变化与统计显著性的关联；

- 热图：展示核心差异基因表达模式聚类，挖掘协同调控特征；

- MA图：辅助判断差异基因分布规律，排除极端表达值干扰。

2.6 核心结果整合与解读

整合所有分析结果与可视化图表，提炼处理因素的基因调控规律，明确差异基因整体特征，为后续功能注释、通路富集及机制研究提供核心靶点。

📊 结果概览

3.1 A. 样本间表达水平

结论：处理组平均基因表达水平显著高于对照组，表明处理对整体基因表达具有明显上调作用，与预期调控效应一致，为差异分析提供基础。

3.2 B. 样本相关性矩阵

结论：对照组（C1-C6）、处理组（T1-T6）组内样本高度相关，说明样本重复性良好、实验设计可靠；组间存在显著表达差异，表明处理因素成功诱导基因表达谱改变。

3.3 C. 差异表达火山图

结论：共筛选出243个差异表达基因，其中上调127个、下调116个（红色=上调，蓝色=下调，灰色=无显著差异）；多数差异基因具有高统计显著性与表达倍数变化，调控作用特异且显著。

3.4 D. 主成分分析（PCA）

结论：两组样本在PCA图中完全分离、无交叉，PC1解释26.3%总变异，是区分两组的主要维度，进一步验证处理对基因表达谱的显著影响，与相关性分析结果相互印证。

3.5 E. 差异表达基因热图

结论：展示表达差异最显著的25个DEGs，组内样本表达趋势一致、组间差异明显，可直观筛选核心调控基因。

3.6 F. MA图：倍数变化与平均表达

结论：呈现基因表达倍数变化（log2FC）与平均表达水平（log10(mean FPKM)）的关联，高/低表达基因中均存在显著差异，分布均匀无表达偏向性，说明处理对不同丰度基因均可能有调控作用。

🎯 综合结论

本研究通过系统的RNA-Seq数据分析与标准化可视化流程，明确处理因素对基因表达的广泛显著调控作用：两组基因表达谱差异显著，筛选出243个DEGs（127个上调、116个下调），样本重复性良好、组间分离明显。多维度可视化结果相互印证，清晰呈现差异基因分布与表达模式，为后续功能富集、调控网络构建及分子机制研究提供坚实基础，同时建立的可视化流程可为同类转录组研究提供参考。

🔧 分析工具与流程

工具清单

1. 数据预处理：Trimmomatic（质量过滤）、FastQC（质量评估）；

2. 表达定量：HISAT2（序列比对）、StringTie（表达量计算）；

3. 差异分析：DESeq2（差异基因筛选）；

4. 可视化：R语言（ggplot2、pheatmap、factoextra包）。

分析流程

原始数据 → 质量控制 → 序列比对 → 表达定量 → 标准化 → 差异分析 → 可视化 → 结果解读
